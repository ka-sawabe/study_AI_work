# テスト仕様書（04_test_specification.md）

## 1. 目的
キッチンタイマーアプリの機能が要件定義通りに動作することを検証する。

## 2. 検証項目

| No | 検証内容 | 手順 | 期待結果 | 結果（机上） | 結果（手動） |
|---|---|---|---|---|---|
| 1-1 | タイマー開始（正常系） | 時間を設定し「スタート」をクリック | カウントダウンが開始される | OK | OK |
| 1-2 | 一時停止（正常系） | カウントダウン中に「ストップ」をクリック | カウントダウンが停止する | OK | OK |
| 1-3 | 再開（正常系） | 停止中に「スタート」をクリック | 停止した時間から再開する | OK | OK |
| 1-4 | リセット（正常系） | 動作中/停止中に「リセット」をクリック | タイマーが0分00秒に戻る | OK | OK |
| 1-5 | タイムアップ（正常系） | 10秒タイマーを設定し終了を待つ | 0分00秒で止まり「タイムアップ！」と表示される | OK | OK |
| 2-1 | 時間未設定での開始（異常系） | 0分0秒の状態で「スタート」をクリック | カウントダウンが開始されない | OK | OK |
| 2-2 | 終了後のボタン操作（異常系） | 「タイムアップ！」表示後に「ストップ」をクリック | 画面の状態が保持される（変化しない） | OK | OK |
