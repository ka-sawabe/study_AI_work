# キッチンタイマー（カウントダウンタイマー）要件定義書

## 1. プロジェクトの目的
ブラウザ上で動作するシンプルなキッチンタイマー（カウントダウンタイマー）を作成します。料理や作業の時間を計測するために使用します。

## 2. 機能要件

### 2.1 タイマー設定機能
- 画面上で「分」と「秒」を指定できます。

### 2.2 タイマー操作機能
以下の3つのボタンでタイマーを操作します。
- **スタート**: 指定した時間からカウントダウンを開始します。一時停止中は停止した時間からカウントダウンを継続します。
- **ストップ**: カウントダウンを一時停止します。
- **リセット**: タイマーを停止し、表示を0分00秒に戻し、入力欄をクリアします。

### 2.3 カウントダウン表示機能
- スタートボタンを押すと、1秒ずつ時間が減っていきます。
- 画面の表示はリアルタイムで更新されます。

### 2.4 終了通知機能
- 0分00秒になったら自動的にカウントダウンが止まります。
- 画面上に「タイムアップ！」と表示し、終了したことをユーザーに分かりやすく伝えます。

## 3. 非機能要件
- **動作環境**: Webブラウザ（特別な環境設定は不要です）。
- **技術スタック**:
    - HTML: 構造の定義
    - CSS: デザインとレイアウト
    - JavaScript: カウントダウン処理とボタン操作（バニラJSを使用）
- **制約事項**: 外部ライブラリは使用せず、単一のフォルダ内で完結する構成とします。
