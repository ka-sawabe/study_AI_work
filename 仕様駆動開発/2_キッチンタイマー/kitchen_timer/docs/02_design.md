# キッチンタイマー 設計書

## 1. 概要
本プロジェクトは、HTML/CSS/Vanilla JavaScriptを用いて、シンプルで直感的に操作できるキッチンタイマーアプリケーションを作成する。

## 2. 画面レイアウト案
画面はシンプルに中央寄せの構成とする。

- **タイトル**: キッチンタイマー
- **表示エリア**: `00:00`（分:秒）
- **入力エリア**: 「分」入力欄（数値のみ）、「秒」入力欄（数値のみ）
- **ボタンエリア**:
    - [スタート] [ストップ] [リセット]
- **メッセージ表示**: 「タイムアップ！」表示用エリア（初期は非表示）

## 3. ファイル構成
```text
kitchen_timer/
├── index.html       # 構造（画面レイアウト）
├── style.css        # デザイン（スタイリング）
└── script.js        # 動作（カウントダウン処理・イベント制御）
```

## 4. クラス・関数設計

今回の構成は小規模であるため、クラス（オブジェクト）を用いたカプセル化を行い、可読性と保守性を高める。

### 4.1 script.js の設計

#### `Timer` クラス
タイマーのロジックを管理するクラス。

- **プロパティ**:
    - `remainingTime`: 現在の残り秒数（整数）
    - `timerId`: setIntervalのID
    - `isRunning`: 動作中かどうか（boolean）

- **主要メソッド**:
    - `start()`: タイマーを開始（または一時停止状態から再開）する。
    - `stop()`: タイマーを一時停止し、isRunningをfalseにする。
    - `reset()`: タイマーを停止し、残り時間を0に戻し、画面表示と入力欄をクリアする。
    - `update()`: 1秒ごとに`remainingTime`を減らし、画面表示を更新する
    - `checkEnd()`: `remainingTime`が0になった場合、停止して終了メッセージを表示する

### 4.2 DOM操作の役割
- `index.html` から各IDを取得し、ボタンのクリックイベントや入力欄の値を `Timer` クラスと連携させる。
- 表示のフォーマット（分:秒）への変換はヘルパー関数で行う。
