# テスト仕様書

## 1. 目的
キッチンタイマーアプリの正常な動作および、入力値などの制限範囲外に対する挙動が要件を満たしているかを確認する。

## 2. 検証項目

| No | 検証内容 | 手順 | 期待結果 | 結果（机上デバッグ） | 結果（手動テスト） |
|---|---|---|---|---|---|
| 01 | 正常なカウントダウン | 分:1, 秒:0 を設定し、スタートを押す | 1分からカウントダウンが始まり、0分00秒で止まる | OK |  |
| 02 | 一時停止 | カウントダウン中にストップを押す | タイマーが停止する | OK |  |
| 03 | 再開 | 一時停止後にスタートを押す | 停止した時間からカウントダウンが継続する | OK |  |
| 04 | リセット | 動作中/停止中にリセットを押す | タイマーが停止し、表示が00:00となり、入力欄が空になる | OK |  |
| 05 | 終了通知 | 0分00秒になる | 「タイムアップ！」と画面に表示される | OK |  |
| 06 | 異常系: 空入力 | 値を入力せずにスタートを押す | 0分00秒のまま、またはエラー表示/警告が出る | OK |  |
| 07 | 異常系: 負の値 | マイナスの値を入力してスタートを押す | カウントダウンが始まらない、またはエラー表示が出る | OK |  |
