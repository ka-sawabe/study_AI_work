# 実装タスク一覧

## 1. ディレクトリとファイルの準備
- [ ] **プロジェクト構造の作成**
    - `kitchen_timer/` フォルダ配下に、`index.html`, `style.css`, `script.js` を作成する。

## 2. UI（HTML/CSS）の実装と基本配置
- [ ] **HTML構造の作成 (index.html)**
    - タイマー表示エリア (`id="display"`) を作成する。
    - 「分」入力エリア (`id="minutes"`)、「秒」入力エリア (`id="seconds"`) を作成する。
    - ボタンエリア（スタート: `id="startBtn"`, ストップ: `id="stopBtn"`, リセット: `id="resetBtn"`）を作成する。
    - 終了通知エリア (`id="message"`) を追加する。
- [ ] **スタイリング (style.css)**
    - 画面中央寄せレイアウトを設定する。
    - 入力エリア、ボタン、表示エリアに適度な余白と視認性を考慮したデザインを適用する。

## 3. ロジック（JavaScript）の基本実装
- [ ] **Timer クラスの実装 (script.js)**
    - `Timer` クラスを定義し、プロパティ `remainingTime`, `timerId`, `isRunning` を初期化する。
    - `update()` メソッドにて、残り時間を1秒ずつ減らし、表示を更新するロジックを実装する。
    - `checkEnd()` メソッドにて、時間が0になった際の停止処理と「タイムアップ！」表示処理を実装する。

## 4. 画面とロジックの結合
- [ ] **イベントリスナーの登録 (script.js)**
    - 「スタート」ボタンのクリック時に、動作中でなければタイマーを開始または一時停止状態から再開する処理を実装する。
    - 「ストップ」ボタンクリック時にタイマーを一時停止する処理を実装する。
    - 「リセット」ボタンクリック時にタイマーを停止、残り時間を0にし、表示と入力欄を初期化する処理を実装する。

## 5. 動作確認（手動テスト）
- [ ] **タイマー機能テスト**
    - 指定した時間（例：1分30秒）でカウントダウンが正しく始まるか確認する。
- [ ] **操作機能テスト**
    - 途中でストップボタンを押し、時間が一時停止することを確認する。
    - リセットボタンを押し、表示が0に戻ることを確認する。
- [ ] **終了通知テスト**
    - 時間が0になったときに、「タイムアップ！」と表示され、カウントダウンが止まることを確認する。
