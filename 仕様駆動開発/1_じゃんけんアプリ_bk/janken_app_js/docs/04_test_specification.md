# じゃんけんゲーム 動作検証仕様書

## 1. 目的
本ドキュメントは、じゃんけんゲームの動作が要件通りに正しく機能しているかを検証するためのテスト項目を定義する。

## 2. 検証項目

| No | 検証内容 | 手順 | 期待結果 | 机上デバッグ | 手動テスト |
|:---|:---|:---|:---|:---|:---|
| 01 | 正常系: グーボタン押下 | グーボタンを押す | 自分の手、コンピュータの手、勝敗結果が正しく表示されること | OK | |
| 02 | 正常系: チョキボタン押下 | チョキボタンを押す | 自分の手、コンピュータの手、勝敗結果が正しく表示されること | OK | |
| 03 | 正常系: パーボタン押下 | パーボタンを押す | 自分の手、コンピュータの手、勝敗結果が正しく表示されること | OK | |
| 04 | 正常系: 5回試行確認 | 5回ゲームを行う | 試行回数が減少し、5回後に合計スコアが表示されること | OK | |
| 05 | 異常系: UI表示確認 | 画面を開く | タイトル、各ボタン、結果表示エリア、残り回数、合計スコアが中央ぞろえで正常に描画されていること | OK | |
| 06 | 異常系: JavaScriptエラー有無 | ブラウザのコンソールを開く | コンソールにエラーメッセージが表示されていないこと | OK | |
