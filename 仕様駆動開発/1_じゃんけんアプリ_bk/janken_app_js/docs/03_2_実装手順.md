# じゃんけんゲーム 実装手順書

## 手順 01: ファイル作成と基本構造の準備
- 目的: プロジェクトに必要な基礎ファイルを揃える。
- 作業:
    - `index.html`: HTML5の雛形を作成し、`style.css` と `script.js` を読み込む設定を追加する。
    - `style.css`: ファイルを空で作成する。
    - `script.js`: ファイルを空で作成する。

## 手順 02: UI（HTML/CSS）の実装
- 目的: ユーザーが操作可能な画面を作る。
- 作業:
    - `index.html`: ボタン（グー・チョキ・パー）と結果表示用のDOM要素（`#result-user`, `#result-computer`, `#result-text`, `#remaining-count`, `#score-board`）を作成する。
    - `style.css`: ボタンのデザインや、結果表示エリアの見た目を整え、画面全体を中央ぞろえにする。

## 手順 03: ロジックのクラス実装 (JavaScript)
- 目的: ゲームの勝敗判定と回数管理を行うエンジンを実装する。
- 作業:
    - `script.js`: `JankenGame` クラスを実装する。
        - `getComputerChoice()` メソッド: コンピュータの手（グー・チョキ・パー）をランダムに決定して返す。
        - `judge(userChoice)` メソッド: ユーザーの手とコンピュータの手を比較し、勝ち・負け・あいこの結果を返す。
        - 試行回数管理と合計スコアの集計ロジックを実装する。

## 手順 04: 画面とロジックの結合
- 目的: ボタン操作に応じてゲームを実行し、画面に結果とステータスを表示する。
- 作業:
    - `script.js`: イベントリスナーを設定し、ボタンクリック時に `JankenGame` クラスの各メソッドを呼び出す。
    - `updateUI(user, computer, result, remaining, score)` メソッド: 判定結果とステータスをHTMLのDOM要素に反映させる。
    - 5回終了後にリセット機能を表示するロジックを追加する。

## 手順 05: 動作確認
- 目的: ゲームが正しく動作することを確認する。
- 作業:
    - 手動で5回ゲームを行い、自分の手・コンピュータの手・勝敗が表示され、残り回数と合計スコアが正しく管理されているか確認する。
