# じゃんけんゲーム 要件定義書

## 1. 概要
ブラウザ上で動作する、コンピュータと対戦するシンプルなじゃんけんゲームです。

## 2. 機能要件
- ユーザーは「グー」「チョキ」「パー」のいずれかを選択するボタンを押すことでゲームを開始します。
- 1回につき連続5回までゲームを行うことができ、ゲーム終了後にプレイヤーの勝敗数（勝ち数・負け数・あいこ数）を表示します。
- システムはランダムにコンピュータの手（グー・チョキ・パー）を決定します。
- ユーザーの選択とコンピュータの結果を比較し、勝敗判定（勝ち・負け・あいこ）を行います。
- 画面上に以下の情報を表示します。
  - 自分の手
  - コンピュータが出した手
  - 勝敗結果
  - 現在の試行回数（残り回数）
  - 合計勝敗数（勝ち数・負け数・あいこ数）

## 3. 技術スタック
- HTML5
- CSS3
- JavaScript（プレーンなバニラJS）
- 外部ライブラリは使用しない構成とします。

## 4. 動作環境
- モダンブラウザ（Chrome, Edge, Firefox, Safari等）で直接ファイルを開くことで動作します。
