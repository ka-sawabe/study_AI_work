# じゃんけんゲーム 要件定義書

## 1. 概要
本プロジェクトでは、Webブラウザで直接動作するシンプルな「じゃんけんゲーム」を作成します。ユーザーは画面上のボタンを選択し、コンピュータと対戦して結果を確認できます。

## 2. 目的
ユーザーが直感的に操作できるじゃんけんゲームを提供し、Webブラウザでの基本的なインタラクションを体験できるようにすること。

## 3. 機能要件
- **プレイヤ操作**: 「グー」「チョキ」「パー」の3つの選択ボタンを表示する。
- **コンピュータの動作**: ユーザーがボタンを押した際、コンピュータはランダムに手を選択する。
- **結果表示**: ユーザーの手、コンピュータの手、および対戦結果（勝ち・負け・あいこ）を画面上に表示する。

## 4. 技術仕様
- **構成**: HTML、CSS、JavaScript（バニラJS）の3ファイル構成（または1つのHTMLファイルにまとめても可）。
- **動作環境**: モダンなWebブラウザ（Chrome, Firefox, Edge, Safari等）で、HTMLファイルを直接開いて動作すること。
- **制限事項**: 外部ライブラリ（React, Vue, jQuery等）は使用しない。

## 5. ユーザー体験 (UX)
- エンジニアではない方でも、見ただけで操作方法がわかるデザインとする。
- ボタンを押すと、即座に結果が反映されるレスポンスの良さを重視する。
