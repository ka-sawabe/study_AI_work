# じゃんけんゲーム 設計書

## 1. 概要
本設計書は、「01_requirements.md」に基づいたじゃんけんゲームのシステム設計を定義する。ブラウザ上で動作するシンプルな構成とし、HTML/CSS/JavaScriptをそれぞれ分離して管理する。

## 2. 画面レイアウト案
- **タイトル**: 「じゃんけんゲーム」
- **操作エリア**: 「グー」「チョキ」「パー」の3つのボタンを横並びに配置。
- **表示エリア**:
    - コンピュータの手を表示する領域。
    - 対戦結果（勝ち・負け・あいこ）を表示する領域。

## 3. ファイル構成
```text
janken_app_js/
├── index.html        # 画面の構造
├── style.css         # スタイリング
└── script.js         # ゲームロジック
```

## 4. クラス・関数設計

### JavaScript (`script.js`)

#### 主要関数
- `playGame(userChoice)`:
    - ユーザーが選択した手を引数として受け取る。
    - コンピュータの手をランダムに決定する（0: グー, 1: チョキ, 2: パー）。
    - ユーザーの手とコンピュータの手を比較し、勝敗を判定する。
    - 結果を画面上の表示エリアに反映させる。

- `getComputerChoice()`:
    - コンピュータの手をランダムに返す関数。

- `determineWinner(user, computer)`:
    - ユーザーとコンピュータの手を受け取り、結果（文字列）を返す関数。
