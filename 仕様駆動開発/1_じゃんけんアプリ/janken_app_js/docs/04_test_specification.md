# じゃんけんゲーム 動作検証仕様書

## 1. 目的
本ドキュメントは、じゃんけんゲームの動作が要件通りに正しく機能しているかを検証するためのテスト項目を定義する。

## 2. 検証項目

| No | 検証内容 | 手順 | 期待結果 | 机上デバッグ | 手動テスト |
|:---|:---|:---|:---|:---|:---|
| 01 | 正常系: グーボタン押下 | グーボタンを押す | コンピュータの手が表示され、勝敗結果が正しく表示されること | OK | OK |
| 02 | 正常系: チョキボタン押下 | チョキボタンを押す | コンピュータの手が表示され、勝敗結果が正しく表示されること | OK | OK |
| 03 | 正常系: パーボタン押下 | パーボタンを押す | コンピュータの手が表示され、勝敗結果が正しく表示されること | OK | OK |
| 04 | 異常系: UI表示確認 | 画面を開く | タイトル、各ボタン、結果表示エリアが正常に描画されていること | OK | OK |
| 05 | 異常系: JavaScriptエラー有無 | ブラウザのコンソールを開く | コンソールにエラーメッセージが表示されていないこと | OK | OK |
