# 実装タスク一覧

## 1. ディレクトリと空ファイルの準備
- [ ] **プロジェクト構造の作成**
    - `janken_app_js` フォルダ内に以下のファイルを空で作成する。
    - `index.html`
    - `style.css`
    - `script.js`

## 2. UI（HTML/CSS）の実装と基本配置
- [ ] **HTML構造の作成 (`index.html`)**
    - タイトルタグの設定。
    - ボタン用エリア（ID: `buttons`）の設定。
    - 結果表示エリア（ID: `result-area`）の設定。
- [ ] **CSSによるスタイリング (`style.css`)**
    - 全体的なレイアウト（中央揃えなど）を整える。
    - ボタンと結果エリアの見た目を調整する。

## 3. ロジック（JavaScript）の基本実装
- [ ] **ゲームロジックの構築 (`script.js`)**
    - コンピュータの手を選択する関数 `getComputerChoice()` を作成。
    - 勝敗を判定する関数 `determineWinner(user, computer)` を作成。
    - 選択肢を定数（グー、チョキ、パー）で定義。

## 4. 画面とロジックの結合（ボタン連携・結果表示）
- [ ] **イベントリスナーの追加 (`script.js`)**
    - HTMLのボタン（ID: `btn-rock`, `btn-scissors`, `btn-paper`）にクリックイベントを設定する。
- [ ] **DOM操作による結果の更新 (`script.js`)**
    - `playGame(userChoice)` 関数内で、判定結果をDOM操作で `result-area` に表示する処理を実装する。

## 5. 動作確認（手動テスト）
- [ ] **ブラウザでの動作確認**
    - `index.html` をブラウザで開き、ボタンをクリックして動作を確認する。
    - ユーザーの手、コンピュータの手、勝敗が表示されるか確認する。
- [ ] **勝敗判定の確認**
    - 全ての組み合わせ（勝ち・負け・あいこ）が正しく判定されるかを確認する。
