# じゃんけんゲーム 実装タスク一覧

## 1. ディレクトリと空ファイルの準備
- [ ] `index.html` ファイルを作成する
- [ ] `style.css` ファイルを作成し、`index.html` に読み込ませる
- [ ] `script.js` ファイルを作成し、`index.html` に読み込ませる

## 2. UI（HTML/CSS）の実装と基本配置
- [ ] `index.html` にタイトル、グー・チョキ・パーのボタンを配置する
- [ ] `index.html` に結果表示用のエリア（自分の手、コンピュータの手、勝敗結果、残り試行回数、合計スコア）を配置する
- [ ] `style.css` でボタンの配置、結果表示が見やすいようにレイアウトを整え、画面全体を中央ぞろえにする

## 3. ロジック（JavaScript）の基本実装
- [ ] `getComputerChoice()` 関数で「グー」「チョキ」「パー」をランダムに返す処理を実装する
- [ ] `determineWinner(user, computer)` 関数で、勝ち・負け・あいこを判定する処理を実装する
- [ ] 最大5回までの試行回数管理と、合計勝敗数を記録するロジックを実装する

## 4. 画面とロジックの結合
- [ ] ボタンにイベントリスナーを追加し、押した手を取得できるようにする
- [ ] ボタンを押した際にロジックを呼び出し、判定結果を自分の手とともに表示エリアに反映させる
- [ ] 5回終了後に結果を表示し、必要に応じてリセット機能を提供する

## 5. 動作確認
- [ ] グー・チョキ・パーをそれぞれ押して、自分の手・コンピュータの手・勝敗が表示されること
- [ ] 試行回数が減少し、5回経過後に合計スコアが表示されること
