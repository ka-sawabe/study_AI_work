# 3. タスク分解のフェーズ
下記に従って「03_tasks.md」を作成してほしい。

- 先ほど作成した「01_requirements.md」と「02_design.md」の内容を前提とすること。

- タスクは以下のフェーズに分けて、上から順番に実行すればアプリが完成するように組み立てること。

- 各チェックリストの項目は、「何をすれば完了（Done）なのか」が明確に判断できる具体的な基準にすること。
