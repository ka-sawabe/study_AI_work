# キッチンタイマーアプリ 設計書

## 1. 概要
本ドキュメントは、要件定義書に基づき、キッチンタイマーアプリの画面レイアウトおよび機能的な設計を記述します。

## 2. 画面レイアウト設計
- タイマー表示エリア: 現在の残り時間（分:秒）を大きく表示します。
- 入力エリア: 分入力ボックス、秒入力ボックスを配置します。
- 操作エリア: 「スタート」「ストップ」「リセット」ボタンを横並びに配置します。
- ステータスメッセージエリア: タイムアップ時に「タイムアップ！」と表示するエリアを設けます。

## 3. ファイル構成
```text
kitchen_timer_vibes/
├── index.html         # アプリの画面構成
├── style.css          # デザイン（見た目）の定義
└── script.js          # タイマーのロジック（動作）
```

## 4. 関数・ロジック設計
バニラJSを使用し、以下の設計で実装します。

### script.js
- `timerInterval`: 現在のタイマーの動作状態を保持する変数。
- `remainingTime`: 現在の残り時間（秒単位）を管理する変数。
- `updateDisplay()`: 画面の表示を更新する関数。
- `startTimer()`: カウントダウンを開始する関数。setIntervalを使用。
- `stopTimer()`: カウントダウンを一時停止する関数。clearIntervalを使用。
- `resetTimer()`: タイマーを停止し、初期状態にリセットする関数。
- `checkTimeUp()`: 0分00秒になったか判定し、「タイムアップ！」を表示する関数。
