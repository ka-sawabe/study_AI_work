# 最初のプロンプト
``` Bash
キッチンタイマーアプリ（カウントダウンタイマー）を、HTML、CSS、JavaScriptで作っていきたい。

■ アプリの要件：
1. 画面に「分」と「秒」を設定できる入力エリア（またはボタン）があること。
2. 「スタート」「ストップ」「リセット」の3つのボタンがあること。
3. スタートを押すと1秒ずつカウントダウンし、画面の表示がリアルタイムで更新されること。
4. 0分00秒になったらカウントダウンが止まり、「タイムアップ！」と画面に分かりやすく表示されること。

■ 作成のルール：
- HTML、CSS、JavaScriptを一通り作成し、ブラウザで動かせる状態にしてほしい。。
- ファイル構成は、index.html、style.css、script.js の3ファイルに分けて作成すること。

- 各コードには、処理の意図がわかる丁寧な日本語コメントを入れること。
 ``` 
